Looping baru
for of vs for in

- for of
  sebuah looping yang bisa menelusuri object yang iterable, apa saja object yang iterable?
  -string
  -array
  -arguments / NodeList
  -TypedArray
  -Map
  -Set
  -User-defined Iterables

- for in
  sebuah looping yang hanya dapat menelusuri key atau value pada objek.
